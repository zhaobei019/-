PHQ-9 的项目反应分析和项目功能检验：基于南京大学本硕学生的压力调查

基于项目反应理论，使用SPSS、R语言对PHQ-9量表进行项目反应分析和功能差异检验，结果发现量表整体和各项目模型拟合良好，项目 5、6、8 在性别上存在功能差异。

电商客服的情绪劳动对离职倾向的影响：工作满意度的中介作用

基于资源保存理论，通过对 123 名一线电商客服员工进行问卷调查研究，探讨情绪劳动对离职倾向的影响，并在此基础上提出一个中介模型，考察工作满意度的中介作用。
